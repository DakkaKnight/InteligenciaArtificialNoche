Inteligencia Artificial y Algoritmos de búsqueda

¿Qué es inteligencia artificial?

La inteligencia artificial se compone de patrones, algoritmos, operaciones y una gran cantidad de información previamente cargada en una base de datos sobre la cual opera esta inteligencia. Actualmente para que esta inteligencia funcione correctamente debe usar estos patrones y algoritmos para dar respuestas a consultas específicas que son dadas por un usuario ajeno a la operación técnica de la misma.

Algoritmos de búsqueda

Estos son una serie de pasos que se deben seguir siguiendo unos patrones específicos para poder encontrar las respuestas o los resultados más óptimos según la consulta propuesta.

¿Cómo pueden Funcionar?

Para este caso vamos a abordar los algoritmos de búsqueda que funcionan siguiendo la teoría de colas, la cual nos sugiere un mecanismo que emplea un modelo que nos sirve para identificar qué "miembro" dentro de una secuencia o lista es el que debe recibir la atención más próxima.

- LIFO - Last In, First Out

Para este caso, la última operación que entra en vigencia es la que se atiende y sale primero del estado.

- FIFO - First In, First Out

Aquí es la primera operación que entró en la cola la que se atiende y sale primero de la lista de atención.

Explicación del Modelo de Red Neuronal

Descripción General

Este código implementa un modelo de red neuronal en TensorFlow para convertir temperaturas de grados Celsius a grados Fahrenheit.

1. Importar las Librerías Necesarias

import tensorflow as tf
import numpy as np
import matplotlib.pyplot as plt

Se utilizan las siguientes librerías:

  • tensorflow: Para construir y entrenar la red neuronal.
  • numpy: Para manejar arrays de datos numéricos.
  • matplotlib.pyplot: Para graficar el error durante el entrenamiento.

2. Definir los Datos de Entrenamiento

celsius = np.array([-40, -10, 0, 8, 15, 22, 38], dtype=float)
fahrenheit = np.array([-40, 14, 32, 46, 59, 72, 100], dtype=float)

Se define un conjunto de datos que contiene los valores de temperatura en grados Celsius y su conversión correspondiente a Fahrenheit.

3. Definir el Modelo de Red Neuronal

oculta1 = tf.keras.layers.Dense(units=3, input_shape=[1])
oculta2 = tf.keras.layers.Dense(units=3)
salida = tf.keras.layers.Dense(units=1)
modelo = tf.keras.Sequential([oculta1, oculta2, salida])

El modelo consta de tres capas:

  • Primera capa oculta con 3 neuronas.
  • Segunda capa oculta con 3 neuronas.
  • Capa de salida con 1 neurona.

4. Compilar el Modelo

modelo.compile(optimizer=tf.keras.optimizers.Adam(0.1), loss='mean_squared_error')

Se utiliza el optimizador Adam con una tasa de aprendizaje de 0.1 y la función de pérdida es el error cuadrático medio (MSE).

5. Entrenar el Modelo

historial = modelo.fit(celsius, fahrenheit, epochs=1000, verbose=False)

El modelo se entrena durante 1000 épocas utilizando los datos de Celsius y Fahrenheit.

6. Graficar la Pérdida Durante el Entrenamiento

plt.plot(historial.history["loss"])
plt.show()

Se muestra una gráfica que representa la pérdida del modelo durante el entrenamiento.

7. Realizar una Predicción

resultado = modelo.predict([100.0])

El modelo predice la conversión de 100 grados Celsius a Fahrenheit.

8. Mostrar las Variables Internas del Modelo

print(oculta1.get_weights())
print(oculta2.get_weights())
print(salida.get_weights())

Se imprimen los pesos y sesgos aprendidos por el modelo durante el entrenamiento.

Resumen

El código construye y entrena un modelo de red neuronal para convertir temperaturas de Celsius a Fahrenheit. Aunque este es un problema simple, el ejemplo sirve para demostrar cómo las redes neuronales pueden aprender relaciones complejas entre datos a través del entrenamiento supervisado.
